# TranslationApp

İndir:
C# WinForms ile geliştirilmiş, MyMemory Translation API kullanan çok dilli metin çeviri uygulaması.

# Özellikler

- 17 farklı dil arasında çeviri
- MyMemory API ile hızlı ve ücretsiz çeviri

- Dil değiştirme (Swap) özelliği

- Ctrl + Enter ile hızlı çeviri

- Temizleme (Clear) butonu

- Detaylı hata yönetimi

- UI kilitleme / açma (işlem sırasında bekletme)

- JSON response işleme

- HTML decode ile temiz sonuçlar
